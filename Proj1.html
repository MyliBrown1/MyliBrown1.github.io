Abandoned Places Project

I started this project when I came across this website while doing research for one of my classes. I wanted to have a list of all of the places on this website along with their location. However, at the time that I accessed this website, there was no option for this. So I decided to use web scraping to create the data that I wanted. The Python code that I used to do that is shown below:

Once the data was cleaned and exported into an excel file, I loaded the data into Tableau to create my visualizations.

I created a Tableau sheet of a map with all of the locations plotted as well as a sheet with a list of the locations.

I combined the two sheets into an interactive dashboard, filtering by whatever location the user clicks. The dashboard also has a Google Search page embedded in it that googles whatever location is clicked on either the map or the table.
